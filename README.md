ARSpatialScanner

An iOS Augmented Reality object scanning application built using SwiftUI, ARKit, Vision, and CoreML.
The app allows users to scan real-world objects using the device camera and detect them using a machine learning model.

This project demonstrates how to integrate AR sessions with real-time ML object recognition while maintaining a clean MVVM architecture.

Demo

Start a scan and point the camera toward objects.
The system analyzes frames in real time and displays detected object information on the screen.

Features

• Real-time object detection using CoreML
• Augmented Reality camera feed powered by ARKit
• Clean architecture using MVVM
• SwiftUI based UI
• Start / Stop scanning control
• Object detection result overlay with dismiss option
• Efficient frame processing using background queues
